## Sugeng Rahayu Booking App ##

Sugeng Rahayu adalah sebuah program untuk booking bus dengan Bahasa pemrograman Python yang dirancang menggunakan konsep Object-Oriented Programming (OOP). Program ini memisahkan logika aplikasi ke dalam dua file utama untuk mempermudah pengelolaan kode:

> sugeng_rahayu.py
File ini berisi kumpulan class yang mengimplementasikan fungsionalitas inti dari sistem perpustakaan.

> sugeng_rahayu_main.py
File ini merupakan interface utama yang memungkinkan pengguna untuk berinteraksi dengan sistem booking app.

====================================================

Cara Penggunaan
Pastikan Python sudah terinstal di komputer Anda. Anda dapat memeriksa dengan perintah berikut di terminal:
python --version

====================================================

Menjalankan Program
> Unduh kedua file (sugeng_rahayu.py dan sugeng_rahayu_main.py) ke komputer Anda
> Buka terminal di direktori tempat file program berada (jadikan 1 folder)
> Jalankan program dengan perintah berikut:
python sugeng_rahayu_main.py
